hexo博客搭建

1.首先配置环境，需要用到git，nodejs，这里google上都有教程
2.安装hexo
执行 npm install hexo-cli -g
可能报错如下：

需要root权限
执行
● sudo npm install hexo-cli -g
● hexo init blog
● cd blog
● npm install
● hexo server
然后访问http://localhost:4000即可访问本地博客，哎呦还不错哦

git迁移

现在我们的博客都只是可以在本地访问，但做好了博客肯定要放出去装下逼啊

1.创建git仓库什么的就不说了，先来看看配置,打开根目录下的_config.yml

2.然后执行

● hexo clean
● hexo generate
● hexo deploy
然后问题又来了

google了很久，终于找到了解决方法
执行下面一条代码后再执行
● npm install hexo-deployer-git –save
